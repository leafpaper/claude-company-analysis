# Executive Summary 字段规范（v4.3 强制）

> Phase 3 写 §一 执行摘要时**必须**按本 schema 的字段名、顺序、必填性展开。禁止自创字段名（如"综合评级" / "量化分" / "定性修正" / "建议仓位"等变体 —— 均为历史漂移产物）。

---

## 7 固定字段（顺序锁定）

| # | 字段名 | 必填 | 格式 | 说明 |
|---|--------|:---:|------|------|
| 1 | **一句话结论** | ✅ | 加粗 + 3 档方向（看多 / 看空 / **中性-分歧**）+ 1-2 句主要逻辑 | 读者扫一眼就能判断观点 |
| 2 | **估值锚** | ✅ | DCF 概率加权的亿元数 / 每股 / 相对当前股价折让% | v4.2 要求:**单锚**（禁止三角验证均值） |
| 3 | **综合评分** | ✅ | `X.XX / 10` + **数据置信度**（高/中/低） | 与 §二 一致,不得独立算数 |
| 4 | **三大风险（Top 3）** | ✅ | 3 条 bullet,每条 ≤ 50 字;优先引用 audit_report 🔴 红旗 | 不多不少 3 条 |
| 5 | **三大机会（Top 3）** | ✅ | 3 条 bullet,每条 ≤ 50 字 | 不多不少 3 条 |
| 6 | **核心非共识观点**（Top 3 洞察标题） | ⚠️ 若 Phase 5 已回写 | 3 条洞察标题(纯标题,不写数学推导) | Phase 3 初版可写"待 Phase 5 回写";Phase 5 完成后必填 |
| 7 | **投资方向综合判定** | ✅ | 3 档之一:**看多方向 / 看空方向 / 中性-分歧** | 与 §十一 综合判断对齐 |

---

## 禁止出现的字段(历史漂移)

以下字段**不得**出现在 §一 执行摘要(已在其他章节承接):

| 禁用字段 | 原因 | 去哪里 |
|----------|------|--------|
| "综合评级"(强烈看好/有条件看好/谨慎/回避 4 档) | v4.1 已改 3 档 | 用第 7 字段"投资方向综合判定" |
| "量化分 + 定性修正 + 调整后分" 多列 | v4.1 禁止打分换壳 | §二 评分总览(单列) |
| "建议仓位: ≤N% 总资产" | 不是"事实",是"建议" | 放 §十 投资回报测算末尾 |
| "尽调优先级 / 需补查问题" | 与 §十四 信息缺口重复 | §十四 |
| "主要假设 / 关键假设敏感度" | 与 §九 DCF 敏感性重复 | §九 |
