# 估值分析框架 (Valuation Analysis Frameworks)

> 基于 Damodaran《Investment Valuation》核心方法论，适配 VC/PE 投资场景。

---

## 一、估值方法选择矩阵

根据公司所处阶段选择适当的估值方法：

| 公司阶段 | 首选方法 | 辅助方法 | 不适用 |
|---------|---------|---------|-------|
| Pre-revenue（种子/天使） | 实物期权 + 可比交易 | Narrative-to-Numbers | DCF |
| 早期收入（A/B轮） | 可比倍数 + Narrative | 实物期权 | 完整 DCF |
| 规模增长（C/D轮） | 简化 DCF + 可比倍数 | Narrative-to-Numbers | — |
| Pre-IPO / 晚期 | 完整 DCF + 可比倍数 | 分部估值 | 实物期权 |

## 二、DCF 简化估值法（适用于 C/D 轮）

### 2.1 操作步骤

1. **确定当前收入基准**：年化收入（最近一个完整年度或最近12个月）
2. **预测 5-7 年收入**：基于三种情景（乐观/基准/悲观增长率）
3. **估计成熟期目标利润率**：参考行业基准（见第 2.3 节）
4. **确定折现率**（股权成本）：

| 折现率构成 | 中国私有公司典型值 | 来源 |
|-----------|----------------|------|
| 无风险利率 | 2.0-2.5%（中国10年国债）| 央行数据 |
| 成熟市场股权风险溢价 | 5.0-6.0% | Damodaran 年度更新 |
| 中国国家风险溢价 | 1.5-3.0% | 基于 CDS 利差和主权评级 |
| 小公司/私有化溢价 | 3.0-5.0% | 规模和流动性风险 |
| **合计股权成本** | **12-16%**（典型区间） | |

*注：分析时应通过联网搜索获取 Damodaran 最新年度数据（damodaran.com）*

5. **计算终值**：使用退出倍数法（比永续增长法更适合私有公司）
   - 终值 = 终年收入（或 EBITDA）× 可比上市公司倍数
   - 对终值应用 **20-30% 流动性折扣**（私有公司 vs 上市公司）
6. **折现至当前**：将各年自由现金流和终值折现求和

| 假设参数 | 乐观 | 基准 | 悲观 | 依据来源 |
|---------|------|------|------|---------|
| 收入增速（CAGR 5年） | {X}% | {X}% | {X}% | {历史增速/行业预测} |
| 目标营业利润率 | {X}% | {X}% | {X}% | {行业基准} |

## 三、相对估值法（可比倍数）

### 3.1 倍数选择指南

| 公司特征 | 首选倍数 | 原因 |
|---------|---------|------|
| 亏损、高增长 | EV/Revenue (PS) | 唯一有意义的指标 |
| 有利润、稳定增长 | EV/EBITDA | 反映运营绩效，排除资本结构和税务差异 |
| 成熟、盈利稳定 | P/E | 盈利可靠可预测 |
| 重资产（硬件/制造） | EV/Invested Capital | 体现资产利用效率 |
| SaaS | EV/ARR | ARR 更好反映经常性收入价值 |
| 平台/市场 | EV/GMV 或 EV/Take Rate Revenue | 适用于交易型平台 |

### 3.2 可比公司筛选标准

**必须满足：**
- 同行业/细分赛道
- 相似商业模式（B2B vs B2C，订阅 vs 交易）
- 相似地域风险（中国 vs 全球）

**尽量匹配：**
- 增速在 1.5 倍以内
- 规模在 5 倍以内
- 利润率趋势相似

**调整因子：**
- 增速溢价：目标增速显著高于可比公司 → 倍数上浮 20-50%
- 规模折扣：目标显著小于可比公司 → 倍数下调 10-30%
- 盈利折扣：目标亏损但可比公司盈利 → 倍数下调 10-20%
- 流动性折扣：私有 vs 上市 → 下调 20-30%

### 3.3 估值三角验证

始终从三个角度交叉验证估值：

| 方法 | 估值区间 | 置信度 | 备注 |
|------|---------|--------|------|
| DCF | {X} - {X} 亿元 | {高/中/低} | {关键假设} |
| 可比倍数 | {X} - {X} 亿元 | {高/中/低} | {可比公司} |
| 最近交易 | {X} 亿元 | {高/中/低} | {本轮/上轮估值} |

**当三者严重分歧时**：
- 最近交易 > DCF 和倍数 → 可能存在估值泡沫
- 最近交易 < DCF 和倍数 → 可能是好的入场时机
- DCF >> 倍数 → 市场对长期前景的定价可能不足
- 必须在报告中解释分歧原因

## 四、实物期权法（适用于早期公司）

### 4.1 适用条件

- 公司有二元结果的关键里程碑（监管审批、技术突破、大客户签约）
- 传统 DCF 产出接近零或负值，但公司明显有价值
- 公司持有"期权"——进入新市场、推出新产品、许可 IP 的能力

### 4.2 定性实物期权框架

由于私有公司缺乏 Black-Scholes 所需的精确输入，使用**定性期权估值**：

| 期权 | 描述 | 成功概率 | 成功后价值 | 行权成本 | 期权价值 |
|------|------|---------|-----------|---------|---------|
| {期权1} | {如：进入车规市场} | {X}% | {X} 亿元 | {X} 亿元 | {概率×(成功价值-成本)} |
| {期权2} | {如：获得某关键认证} | {X}% | {X} 亿元 | {X} 亿元 | {同上} |
| {期权3} | {如：技术授权/IP变现} | {X}% | {X} 亿元 | {X} 亿元 | {同上} |
| **期权总价值** | | | | | **{求和}** |

**公司总估值 = 基础业务价值（DCF/倍数）+ 实物期权价值**

### 4.3 使用注意

- 实物期权容易被高估——对成功概率保持保守
- 期权价值不应超过基础业务价值的 50%（除非公司本质是一个期权，如管线期生物医药）
- 时间衰减：期权窗口越短，价值越低

## 五、Narrative-to-Numbers 框架

### 5.1 方法论（来自 Damodaran）

**核心理念**：每一个估值都是一个故事。好的估值是把故事翻译成数字，然后检验故事的脆弱性。

**操作步骤：**

1. **写故事**（2-3 段）：用非技术语言描述公司的未来——它会成为什么？为什么？
2. **逐句翻译**：

| 叙事 | 数字翻译 | 来源 |
|------|---------|------|
| "巨大的市场" | TAM = {X} 亿美元 | {研究报告} |
| "高速增长" | 收入 CAGR = {X}% | {历史数据/预测} |
| "规模化后高利润" | 目标营业利润率 = {X}% | {行业基准} |
| "X年后上市" | 退出倍数 = {X}x | {可比IPO} |

3. **找到最脆弱假设**：如果哪个假设错了，整个故事就崩塌？
4. **对最脆弱假设做敏感性分析**

### 5.2 输出模板

**公司叙事**：{2-3 句话}

**数字翻译**：{表格}

**最脆弱假设**：{指出最关键的 1-2 个假设}

**如果故事错误**：{估值变为多少？回报变为多少？}

## 六、估值输出规范

### 6.1 必须输出项

1. **使用的估值方法**及选择理由
2. **DCF 假设表**（如适用）及每个假设的来源
3. **可比公司表**及调整因子
4. **估值三角验证表**
5. **Narrative-to-Numbers 摘要**
6. **最终估值判断**：当前估值是 合理 / 偏高 / 偏低 / 严重高估 / 严重低估

### 6.2 估值与评分的关系

- 估值分析是独立章节，**不直接修改综合评分**
- 但估值判断会影响 **维度 9（融资历史与估值）** 的评分：
  - 当前估值 < DCF/倍数中位数 → 维度 9 可加分
  - 当前估值 > DCF/倍数中位数 50%+ → 维度 9 应减分
- 估值判断也影响 **Phase 6 投资回报模拟** 的入场估值假设
