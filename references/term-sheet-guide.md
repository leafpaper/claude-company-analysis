# 条款分析指南 (Term Sheet Analysis Guide)

> 基于 Brad Feld《Venture Deals》核心框架，分析融资条款对投资回报的影响。

---

## 一、关键经济条款

### 1.1 估值与稀释

**基本概念：**
- Pre-money 估值 = 融资前公司价值
- Post-money 估值 = Pre-money + 本轮融资额
- 投资人持股 = 投资额 ÷ Post-money

**期权池陷阱：**
如果本轮要求在 **pre-money** 基础上创建/扩大期权池，实际上是从现有股东（创始人）稀释，而非从新投资人稀释。

| 场景 | Pre-money | 期权池 | 投资额 | Post-money | 创始人实际稀释 |
|------|----------|--------|--------|-----------|-------------|
| 无期权池 | 10亿 | 0 | 2亿 | 12亿 | 16.7% |
| 10% pre-money 期权池 | 10亿（含1亿期权） | 1亿 | 2亿 | 12亿 | 25.0% |

**红旗信号：** C/D 轮要求创建 >15% 的 pre-money 期权池 = 隐藏的估值降低

**Fully Diluted 分析**：
始终在 fully diluted 基础上计算持股——包括所有已发行股份、已授予期权、未行权期权、可转债/SAFE 转换后股份。

### 1.2 优先清算权 (Liquidation Preferences)

| 类型 | 机制 | 投资人友好度 | 中国市场常见度 |
|------|------|-----------|-------------|
| **1x Non-participating** | 先拿回 1x 投资额 **或** 按比例分配（取高者） | 市场标准 | 最常见 |
| **1x Participating** | 先拿回 1x 投资额 **且** 参与按比例分配 | 偏投资人 | 次常见 |
| **Participating + Cap** | Participating 但有回报上限（如 3x） | 中等偏投资人 | 偶见 |
| **>1x Preference** | 先拿回 2x 或 3x 投资额 | 危险信号 | 少见但需警惕 |

**退出瀑布示例**（1x Participating vs 1x Non-participating）：

假设：投资人 A 投资 2 亿元，占 20%。公司以 15 亿元退出。

| | Non-participating | Participating |
|---|------------------|--------------|
| A 的 1x 优先 | 2 亿 | 2 亿 |
| 剩余分配 | 13 亿 × 20% = 2.6 亿 | 13 亿 × 20% = 2.6 亿 |
| A 选择 | max(2, 2.6) = **2.6 亿** | 2 + 2.6 = **4.6 亿** |
| A 回报 | **1.3x** | **2.3x** |

**关键影响**：在中低退出估值场景下，participating preferred 显著侵蚀普通股东（包括后轮投资人转换后）的回报。

### 1.3 反稀释条款 (Anti-Dilution)

| 类型 | 机制 | 对创始人影响 |
|------|------|-----------|
| **Broad-based Weighted Average** | 根据新旧股份加权平均调整转换价格 | 市场标准，影响温和 |
| **Narrow-based Weighted Average** | 同上但分母更小（只算优先股） | 对创始人稍不利 |
| **Full Ratchet** | 转换价格直接降至新一轮价格 | **危险信号** — 在 down round 时极度稀释创始人 |

**何时触发**：仅在 Down Round（新一轮价格低于上一轮）时触发。

### 1.4 其他经济条款

| 条款 | 说明 | 标准 vs 非标准 |
|------|------|-------------|
| **Dividends** | 累计 vs 非累计优先股股息 | 非累计 = 标准；累计 = 注意 |
| **Pay-to-Play** | 不跟投的投资人丧失优先权 | 对公司有利（逼投资人持续支持） |
| **Redemption Rights** | 投资人可在 X 年后要求公司回购股份 | 在中国较常见，注意触发条件 |

## 二、控制条款

### 2.1 董事会构成

| 构成 | 评估 |
|------|------|
| 2 创始人 + 2 投资人 + 1 独立 | 健康平衡 |
| 创始人多数 | 对创始人有利（早期常见） |
| 投资人多数 | **C/D 轮警示信号**（限制创始人自主权） |
| 多个投资人各占 1 席 | 决策效率可能降低 |

观察员席位（observer seat）：比投票席弱，但仍有信息权和影响力。

### 2.2 保护性条款 (Protective Provisions)

**标准/可接受：**
- 否决公司出售/合并
- 否决新股发行（防止不当稀释）
- 否决修改公司章程
- 否决超额举债

**过度/需谈判：**
- 否决年度预算
- 否决高管招聘/解聘
- 否决进入新市场
- 否决日常经营决策

### 2.3 拖带权与随售权

| 条款 | 含义 | 关注点 |
|------|------|--------|
| **Drag-Along（拖带权）** | 多数股东可强制少数股东一起出售 | 触发门槛是多少？（50%? 67%? 75%?） |
| **Tag-Along（随售权）** | 少数股东可在多数出售时一起卖 | 保护小股东的基本权利 |

## 三、中国特色条款

### 3.1 对赌条款（Valuation Adjustment Mechanism / VAM）

中国 VC 市场的独特条款，国际通行的 term sheet 中罕见。

| 对赌类型 | 含义 | 风险等级 |
|---------|------|---------|
| **业绩对赌** | 未达业绩目标需补偿股份或现金 | 高（可能反向稀释创始人） |
| **IPO 对赌** | 约定期限内未上市需回购 | 高（造成上市压力） |
| **估值调整** | 实际估值低于约定时调整 | 中 |

**评估要点**：
- 对赌目标是否合理？（与公司实际增速对比）
- 补偿方式：股份还是现金？（现金对赌压力更大）
- 是否有上限？（无上限 = 危险）
- 对赌主体：公司还是创始人个人？

### 3.2 回购条款

投资人有权在特定条件下要求公司或创始人回购股份。

**常见触发条件**：
- 约定期限内未完成 IPO
- 重大违约或业绩严重不达标
- 创始人离职

**评估**：回购条款实质上将股权投资变成了有退出保障的债权——对投资人有利，但可能给公司造成现金压力。

## 四、可转债与 SAFE

### 4.1 关键条款

| 条款 | 典型值 | 说明 |
|------|--------|------|
| 折扣率 (Discount) | 15-25% | 转股时相对新一轮价格的折扣 |
| 估值上限 (Cap) | 因公司而异 | 转换估值的上限，比折扣更重要 |
| 利息 | 2-8% | 简单利息，通常转为股份 |
| 到期日 | 12-24 个月 | 到期未转换时的处理方式 |
| 转换触发 | 合格融资/上市/出售 | 什么事件触发转换 |
| MFN 条款 | 有/无 | 后续发行更优惠的可转债时自动获得相同条款 |

### 4.2 Cap Table 影响分析

如果公司有未转换的可转债/SAFE，必须计算转换后对 cap table 的影响：

1. 确定所有未偿可转债/SAFE 的金额和条款
2. 假设在本轮估值下转换，计算转换后的额外股份
3. 重新计算 fully diluted 持股比例
4. 如果有多个不同 cap 的 SAFE → 按各自 cap 分别计算

## 五、条款友好度评分

### 5.1 五星评估

| 评分 | 含义 | 特征 |
|------|------|------|
| ★★★★★ | 标准/偏公司友好 | 1x non-participating, broad-based WA, 平衡董事会, 无对赌 |
| ★★★★ | 基本标准 | 大部分标准，1-2 个条款略偏投资人 |
| ★★★ | 可接受但需关注 | 若干条款偏投资人（如 participating preferred 或轻度对赌） |
| ★★ | 需谨慎评估 | 多个条款偏投资人，对赌目标偏高，或 >1x 优先清算权 |
| ★ | 危险信号 | Full ratchet 反稀释、投资人控制董事会、极端对赌、高倍优先清算权 |

### 5.2 条款检查清单

| # | 检查项 | 状态 | 评估 |
|---|--------|------|------|
| 1 | 估值是否合理？（vs 可比公司同阶段融资） | ☐ | |
| 2 | 期权池大小和创建时间（pre vs post money）？ | ☐ | |
| 3 | 优先清算权类型和倍数？ | ☐ | |
| 4 | 反稀释条款类型？ | ☐ | |
| 5 | 董事会构成变化？ | ☐ | |
| 6 | 有无对赌条款？目标是否合理？ | ☐ | |
| 7 | 有无回购条款？触发条件？ | ☐ | |
| 8 | Pro-rata 权利是否包含？ | ☐ | |
| 9 | 已有可转债/SAFE 的 cap table 影响？ | ☐ | |
| 10 | 创始人锁定/竞业/知识产权归属？ | ☐ | |

## 六、条款对回报的影响建模

### 6.1 退出瀑布分析模板

在投资回报模拟中，应分别建模"无条款影响"和"含条款影响"的回报：

| {基准} | {X} | {X} | {X} | {X}x |
| {悲观} | {X} | {X} | {X} | {X}x |

### 6.2 关键发现

条款影响在**低退出估值**时最显著——当退出估值仅为入场估值的 1-2x 时，participating preferred 和高倍优先清算权可以让投资人获得远超按比例计算的回报，而普通股东（包括后轮投资人）被显著压缩。

**因此**：在悲观情景中，条款分析比乐观情景更加重要。
